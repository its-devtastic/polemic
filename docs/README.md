# Getting started

## Installation
```shell
npm install -g polemic
```

## Usage

### Create

Creates a new Polemic project based on a template

```shell
polemic create principia --template=book
```

### Preview

Starts a live preview of your project.

```shell
polemic preview
```

### Export

Exports your Polemic project to a static HTML website.

```shell
polemic export
```
